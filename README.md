# Sistema de Reconhecimento Facial Offline - Portaria

Este projeto permite o **reconhecimento facial offline** para abertura de portas, pensado para **pessoas com deficiência visual**. Todo o sistema roda em **um único computador com câmera USB**.

## Funcionalidades
- Cadastro de usuários com fotos capturadas pela câmera.
- Reconhecimento facial offline usando `face-api.js`.
- Emissão de som quando o usuário é reconhecido.
- Estrutura simples com Node.js + Express.
